# 📊 Sales Analytics Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive Power BI dashboard designed to analyze sales performance across different segments, categories, regions, and shipping modes.

## 🚀 Key Features
- Segment-wise Sales Analysis
- Category & Sub-Category Profit Insights
- Region-wise Sales Distribution
- Ship Mode Quantity Analysis
- Dynamic Filters (Date, City)

## 🛠 Tools Used
- Power BI
- DAX
- Data Cleaning

## 📈 Insights
- Consumer segment drives the highest revenue
- Office Supplies is the most profitable category
- Standard shipping is the most used mode

## 📂 Files Included
- Power BI Dashboard (.pbix)
- Dataset (.csv)

## 🎯 Conclusion
This dashboard helps businesses track performance, identify trends, and make data-driven decisions.
